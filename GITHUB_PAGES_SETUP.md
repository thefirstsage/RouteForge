# GitHub Pages Setup

Use GitHub Pages to publish RouteForge legal documents, including the Google Play privacy policy URL.

## Steps

1. Go to the GitHub repo.
2. Open **Settings**.
3. Open **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Set **Branch** to **main**.
6. Set **Folder** to **/docs**.
7. Click **Save**.
8. After GitHub generates the Pages URL, use the privacy policy page URL for Google Play.

The privacy policy URL should look similar to:

```text
https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/privacy-policy.html
```

If the repo uses a project page path, GitHub may include the repo name in the URL.
