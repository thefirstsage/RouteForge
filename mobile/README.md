# RouteForge Mobile

The mobile MVP lives in `mobile/routeforge-mobile`.

## Run

Open a terminal in `mobile/routeforge-mobile` and run:

```powershell
npm.cmd install
npm.cmd run start
```

Then use Expo Go, an Android emulator, an iOS simulator, or Expo web depending on your local setup.

## Current Scope

- Mock business generation
- Local saved routes with AsyncStorage
- Hidden / not interested business suppression
- State selection with last selected state saved locally
- Expanded Today’s Route stop tracking
- Optional follow-up date capture
- External Google Maps links
- No login, billing, dashboards, embedded maps, or desktop migration
