# RouteForge

**Plan your route. Knock more doors. Close more deals.**

RouteForge is a simple tool for local service businesses to:

* Find nearby businesses
* Build a daily route
* Track follow-ups
* Stay organized and consistent

---

## 🚀 Download (Windows)

👉 Download here: https://github.com/thefirstsage/RouteForge/releases/tag/v0.1.0

> ⚠️ Windows may show a SmartScreen warning (normal for new apps).
> Click “More Info” → “Run anyway”.

## 🧠 What this is for

If you:

* do door knocking

* run a service business (window cleaning, pressure washing, etc.)

This is built to help you get more clients without guessing.
